# 🔥 Streak — Goal Tracker

A beautiful **daily streak tracking web app** that helps you stay consistent with your goals.
Track habits, build streaks, and keep your motivational **fire streak alive** day after day.

The app is designed to be **simple, fast, and visually satisfying**, featuring animated flames that grow stronger as your streak increases.

---

# ✨ Features

## 🔥 Animated Streak Flame

* Flame grows visually as your streak increases.
* Different flame intensity levels:

  * **0 days** → dormant flame
  * **1–9 days** → small flame
  * **10–19 days** → larger flame
  * **20–29 days** → brighter flame
  * **30–49 days** → glowing flame
  * **50+ days** → powerful flame with sparks

## 📅 Interactive Calendar

* Click any day to mark it completed.
* Completed days are highlighted.
* Current streak days are visually connected.
* Today's date is automatically highlighted.

## 🎯 Multiple Goals

Create and manage multiple goals simultaneously.

Each goal includes:

* Title
* Optional description
* Custom start date
* Independent streak counter

## ✔ One-Click Daily Completion

Mark today complete with one button:

